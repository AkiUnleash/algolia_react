# Algolia-React

## 概要

Algolia を体験するためのテンプレート

## 内容

- Algolia
- React
- Typescript
- Redux-toolkit
- Webpack
- Sass
- Dotenv

## API キーの設定

[.env]ファイルを作成し、Algolia の API キーを以下のように保存してください。

```
ALGOLIA_APPLICATION_ID="xxxxxxxxx"
ALGOLIA_SEARCH_ONLY_API="xxxxxxxxxxxxxxxx"
ALGOLIA_ADMIN_API="xxxxxxxx"
ALGOLIA_INDEX="xxxxx"
```

## 使い方

ローカルサーバーの起動

```
yarn start
```

ビルド

```
yarn build
```
